Mathematical uint64_t reverse_integer ( uint64_t n,
unsigned base )

Reverses the digits of the number n. Ignores lead zeroes, works in any base.

Example:

reverse_integer( 0xFA, 16 ) // => 0xAF
reverse_integer( 0b0101, 2 ) // => 0b101
